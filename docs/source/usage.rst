Usage
=====

Comenzar y activar un entorno virtual
--------------------------------------

.. code-block:: console

    $ python -m venv venv

    $ .\venv\Scripts\activate (windows)
    $ source venv\bin\activate (linux)



Instalación
------------

Intala las siguientes librerías

.. code-block:: console

   (.venv) $ pip install django



Iniciar el servidor
--------------------

.. code-block:: console

   (.venv) $ python3 manage.py runserver
