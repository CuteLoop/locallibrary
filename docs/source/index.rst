¡Bienvenido a la documentación de LocalLibrary!
=================================================

**LocalLibrary** es una librería de python escrita django que extiende la versión de `Mozilla <https://developer.mozilla.org/en-US/docs/Learn/Server-side/Django/Tutorial_local_library_website>`_
 Esta escrito para ser usado como software para administrar una *biblioteca*.

.. note::

   Este proyecto se encuentra bajo desarrollo.
